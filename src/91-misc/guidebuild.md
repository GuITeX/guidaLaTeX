# Produzione della guida

Questa guida è costruita con [`mdbook`](https://rust-lang.github.io/mdBook/) a
partire da file testuali in formato Markdown.

I file del progetto sono tenuti sotto controllo di versione tramite `git` e resi
pubblici in [questo repository
Github.com](https://github.com/GuITeX/guidalatex).
